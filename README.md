# JavaScript Basics - Day 4 Internship

This project was completed as part of my Day 4 Internship Training.

## Concepts Covered

- Conditional Statements (if, else if, else)
- Variable Swapping
- Functions
- Objects and Methods
- Arrow Functions
- Pass/Fail Result Evaluation

## Features

### 1. Number Check
Determines whether a number is:
- Positive
- Negative
- Zero

### 2. Variable Swapping
Swaps the values of two variables using a temporary variable.

### 3. Largest Number
Finds the largest among two numbers using a function.

### 4. Student Object
Stores student details using an object and displays them using a method.

### 5. Result Evaluation
Uses an arrow function to determine whether a student has passed or failed based on marks.

## Technologies Used

- HTML5
- JavaScript (ES6)

## Output

- Number status
- Swapped variable values
- Largest number
- Student details
- Pass/Fail result
